Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. The most famous are these as follows getElementsByTagName, getElementsByClassName, getElementById, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML reads both the html tag and text inside the element and the main difference between innerText and textContent is that the innerText only shows the text that is rendered on the screen and textContent shows all the text inside the elements markup including its childrens and hidden texts.

Question-3: What is event delegation in the DOM?

Its a technique where an event is added to a parent element. upon clicking on any of its child it delegates events to its child. using this technique we wont have to attach events to all of its childrens.

Question-4: What is event bubbling in the DOM?

Event bubbling is the default behavior when there are multiple child elements are nested inside a parent element and parent and most of the child has events attached to them are triggered all at once, these events executes in a hierarchy of child to parent approach like bubbling up, hence the name event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

Using JavaScript we can create element with createElement() method, and to add and remove elements from DOM tree we can use methods like appendChild() and removeChild().
